# 🚗 Car Racing Game in Python

A simple and exciting Car Racing Game developed using Python and Pygame. Control your car, avoid obstacles, and score as many points as possible while surviving on the road.

## Features

* Smooth car movement
* Obstacle avoidance
* Score tracking
* Collision detection
* Simple and interactive gameplay
* Beginner-friendly Python project

## Technologies Used

* Python
* Pygame

## Installation

1. Clone the repository:

```bash
git clone https://github.com/muhammadabdurrehman-arif/Car-Racing-Game-in-python.git
```

2. Install dependencies:

```bash
pip install pygame
```

3. Run the game:

```bash
python main.py
```

## Controls

* Left Arrow → Move Left
* Right Arrow → Move Right
* Esc → Exit Game
